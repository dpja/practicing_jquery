.toggleClass( className )

TYPE :STRING

Add or remove one or more classes from each element in the set of matched elements, depending on either the class's presence or the value of the state argument.

.toggleClass( className,state )

TYPE :STRING One or more class names (separated by spaces) to be toggled for each element in the matched set. TYPE :state A Boolean (not just truthy/falsy) value to determine whether the class should be added or removed.

toggle background between steelblue and green
